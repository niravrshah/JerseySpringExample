    Welcome to JerseySpringExample!

    This project demonstrates the simple implementation of provisioning your REST based Web Service infrastructure using the best breeds of libraries.

    What's Included

    1. Jersey - The reference implementation of the JAX-RS from Oracle. Provides a very rich, and robust API for implementing RESTful Services, including a client Library
    2. Utilize the Faster-XML and Hibernate object mapper to remove the JSON circular dependencies issue
    3. Demonstrate different reference examples regarding building different CRUD based entity service (BaseService, BaseCrudEntityService etc..)
    4. Jersey Client utility with sample code which could help to make a system to system call to REST resources
    5. Provide Jersey Jackson + Spring integration
    6. Demonstrate an example to build the REST based CRUD services
    7. Web Service Discovery using WADL implementation

    How to Build?

    We are using maven to build and manage the project dependencies. To build the project, go to the project work directory and execute below maven goal

    mvn clean install

    Handy URIs after deployment. You could check the currently deployed version to EC2 by clicking on them below.

    Below sample URIs will be helpful to you to navigate some of the REST based service comes free with this project and ready to use after building and deploying the project. [Currently deployed in EC2]

    1. WADL
    2. Get All Sources
    3. Get Source ID by Source name
    4. Get Source name by Source ID
    5. Update payment
    6. Persist parent/child cascading through SOAP-UI/rest client ONLY(also remove JSON circular dependencies)
